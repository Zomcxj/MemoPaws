# 桌面效率工具 - 待开发功能清单

> 最后更新：2026-06-13

---

## 一、备忘录模块

### 1.1 自定义存储路径
- 支持用户自定义备忘录文件存储文件夹路径
- 默认路径为 `C:\Users\<用户名>\Documents\memo`
- 修改路径后提供一键迁移功能，自动将原有文件剪切到新路径并更新索引

### 1.2 Markdown 语法高亮
- 编辑/预览模式下对 Markdown 语法进行高亮渲染
- 标签和普通文本使用不同颜色区分

### 1.3 标签分类功能
- 支持为单条笔记添加多个标签
- 支持批量打标签
- 搭配搜索框按标签快速筛选定位笔记
- 新增关键词自动打标签功能，根据笔记内容自动推荐/分配标签

### 1.4 Obsidian 知识库关联
- 支持关联本地 Obsidian vault 文件夹
- 实现备忘录与 Obsidian 的双向同步
- 笔记以标准 Markdown 格式存储，兼容 Obsidian 读取

### 1.5 草稿自动保存
- 编辑中的笔记自动保存草稿，防止意外丢失

## 二、剪贴板模块

### 2.1 图片复制支持
- 复制图片后生成缩略图存入剪贴板历史记录
- 支持两种存储模式：
  - **临时缓存**：图片仅保留一段时间后自动清理
  - **永久存储**：图片长期保存在历史记录中

### 2.2 按内容类型筛选
- 支持按内容类型（文本、图片、代码等）筛选历史记录

### 2.3 自动整理重复内容
- 自动检测剪贴板中的重复内容并进行合并/标记

---

## 三、截图识别模块

### 3.1 实时像素尺寸显示
- 截图选区时实时显示选区的像素尺寸（宽×高）
- 支持先大概框选，再通过输入精确像素值调整选区大小
- 提供应用按钮确认尺寸调整

### 3.2 迷你快捷弹窗
- 截图完成后弹出迷你快捷操作弹窗，可直接在弹窗内完成：
  - 尺寸调整
  - 文字识别（OCR）
  - 翻译
  - 一键复制文本
- 默认不强制保存截图，保留单独的保存按钮
- 主界面保留历史记录管理功能，自动同步弹窗操作结果

### 3.3 系统右键菜单 OCR 提取
- 在系统右键菜单中添加 OCR 提取选项
- 可直接识别本地文件夹或剪贴板中图片的文字内容
- 支持绕过网页/软件的复制限制，提取不支持划词复制区域的文本

### 3.4 批量 OCR 文字提取
- 支持批量选择多张图片进行 OCR 文字提取

### 3.5 延迟截图
- 支持设置倒计时延迟截图（如 3秒、5秒、10秒）

### 3.6 一键复制识别文本
- 截图识别翻译界面中，识别文字区和翻译文字区各新增一键复制按钮

### 3.7 预设常用截图尺寸模板
- 提供常用截图尺寸预设（如 1920×1080、1280×720、1080×1920、800×600 等）
- 支持自定义添加预设尺寸

## 四、加密密钥管理模块（新增）

### 4.1 基本功能
- 用于加密存储账号密码、API 密钥等私密信息
- 需主密码验证解锁才能查看内容
- 支持添加、编辑、删除密钥条目
- 每条密钥包含：名称、账号/密钥值、过期时间、分类图标等

### 4.2 状态栏快捷入口
- 在系统状态栏添加带锁标的独立快捷按钮
- 方便快速调用密钥管理功能

### 4.3 密钥过期提醒
- 支持设置密钥过期时间
- 即将过期的密钥显示黄色警告标识
- 已过期的密钥显示红色警告标识
- 支持按过期状态筛选密钥列表

---

## 五、全局优化

### 5.1 自定义快捷键
- 支持自定义各功能模块的全局快捷键
- 包括：截图、快速打开剪贴板、新建备忘录、密钥管理快捷入口等

### 5.2 快速启动快捷功能
- 支持通过快捷键快速启动对应功能模块

### 5.3 文本替换工具
- 支持自定义快捷输入模板（文本替换规则）
- 输入缩写自动替换为预设的完整文本
- 支持添加、编辑、删除替换模板
